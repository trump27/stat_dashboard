## 利用

- ``dataフォルダ``を作成する
```
> python call_api.py
```
- 月単位のデータを取得する
- 開始日時は ``yyyymm00``形式。現在までのデータを取得する
- 取得したjsonファイル、日付と値のcsvファイルをIndicatorCode単位に一括出力する
- APIキー、認証などは不要
